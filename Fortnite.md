^ = Ctrl
M = Alt (or meta key)

(note: some customs bindings)

## MOVE

```
W         : Up
S         : Down
A         : Left
D         : Right
Space     : Jump
Shift     : Sprint
=         : Auto run
Left Ctrl : Crouch (tap) / Slide (hold)
,         : Force jog
```

## COMBAT

```
Left mouse  : Fire
Mid mouse   : Target
Right mouse : Reload
E           : Use
1           : Harvest tool
2 - 6       : Weapons 1-5

Mousewheel up   : Build wall
Mousewheel down : Build stairs
Mouse thumb 1   : Weapon 1
Mouse thumb 2   : Weapon 2
```

## BUILD

```
Left Ctrl  : Build & crouch
F1         : Wall
F2         : Floor
F3         : Stairs
F4         : Roof
F5         : Trap
Left mouse : Place building
F          : Repair / upgrade
R          : Rotate
Right mouse: Change build material
G          : Build edit
Left Ctrl  : Edit & crouch
Left mouse : select building edit
Right mouse: Reset building edit
```

## COMMUNICATION

```
Y    : Push to talk
V    : Squad comms
]    : Ping / Place marker
[    : Place enemy marker
```

## MISC

```
B        : Emote
L        : Toggle harvesting tool
T        : Trap equip / picker
Q        : Switch quickbar
Left Alt : Cursor mode
M        : Toggle map
Tab      : Toggle inventory
F        : DNBO/Oject carry/place
```

## VEHICLES GENERIC

```
Left shift    : Boost
E             : Unhook / Exit
Right mouse   : Passenger fire
Left mouse    : Fire
Space         : Driver boost / slide / push
```
