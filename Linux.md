Keyboard shortcuts for Linux

### programs
* `Ctrl + Alt + Enter`  - Terminal
* `Ctrl + Alt + T`  - Terminal
* `Ctrl + Alt + B`  - Browser
* `Ctrl + Alt + H`  - Home folder
* `Ctrl + Alt + K`  - Keepass
* `Alt + F4`  - Close program

### System
* `Win + L` - Lock screen
* `Ctrl + Alt + Delete`  - Shutdown system
